## 核心模組

### MCP 閘道（mcp-gateway）

MCP 閘道為系統核心組件，負責處理所有 MCP 協定請求：

<CardGroup cols={2}>
  <Card title="請求處理" icon="globe">
    - **統一入口**：`/*` 監聽所有 HTTP 請求
    - **路由分配**：根據路徑前綴動態分發請求
  </Card>
  <Card title="協定轉換" icon="arrows-spin">
    - **協定解析**：解析 JSON-RPC 格式請求
    - **工具分配**：解析工具名稱並建構呼叫參數
  </Card>
</CardGroup>
