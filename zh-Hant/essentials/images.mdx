## 圖片

### 使用 Markdown

[Markdown 語法](https://www.markdownguide.org/basic-syntax/#images)可用下列方式插入圖片：

```md
![標題](/path/image.jpg)
```

注意圖片檔案大小需低於 5MB。否則建議使用 [Cloudinary](https://cloudinary.com/) 或 [S3](https://aws.amazon.com/s3/) 等服務，便於嵌入外部網址。

### 使用嵌入

如需更多自定義，可使用 [embed](/writing-content/embed) 加入圖片：

```html
<img height="200" src="/path/image.jpg" />
```
