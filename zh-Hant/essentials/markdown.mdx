## 標題

最佳用於區段標頭。

```md
## 標題
```

### 子標題

最佳用於小節標頭。

```md
### 子標題
```

<Tip>

每個 **標題** 和 **子標題** 都會建立錨點，也會顯示在右側目錄中。

</Tip>

## 文字樣式

大多數 Markdown 標記皆有支援。僅需於文字前後加上 `**`、`_` 或 `~` 即可套用樣式。

| 樣式         | 寫法              | 顯示效果         |
| ------------ | ----------------- | ---------------- |
| 粗體         | `**bold**`        | **bold**         |
| 斜體         | `_italic_`        | _italic_         |
| 刪除線       | `~strikethrough~` | ~strikethrough~  |

亦可組合使用，例如撰寫 `**_粗斜體_**` 可得到 **_粗斜體_** 文字。
